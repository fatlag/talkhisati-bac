App Name: تلخيصاتي-بكالوريا
Version: v1.0

Description:
The "تلخيصاتي-بكالوريا" app is a free educational application for third-year high school students (Baccalaureate) in Algeria. It contains summaries, exercises, and tests in PDF format. All content has been collected from official and semi-official Algerian educational websites with the aim of facilitating the collection of resources in one place.

Features:

Easy access to study summaries
Interactive exercises
PDF tests for exam preparation
